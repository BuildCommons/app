# BuildCommons

## Getting started

Run this BOS gateway at [127.0.0.1:8080](http://127.0.0.1:8080) to render your widgets locally.

1. Install packages

```cmd
yarn install
```

2. Start workspace

```cmd
yarn run dev
```

Main entry point for this app: `buildcommons.near/widget/app`
